# Scale Defect Cause Analysis in Steel Rolling Process (POSCO)  
AI BIGDATA Academy 25th A4 Group Project  

## Project Introduction  
This project focuses on identifying the root causes of scale defects in the steel rolling process and proposing data-driven process optimization strategies.  
By analyzing large-scale manufacturing data, we aim to improve product quality and enhance process efficiency.

## Methodology  
1. Data Preprocessing  
   - Handled missing values and outliers  
   - Generated derived variables based on domain knowledge  

2. Feature Selection  
   - Conducted statistical tests to remove insignificant variables  
   - Addressed multicollinearity issues  

3. Data Analysis  
   - Applied multiple linear regression (OLS)  
   - Performed residual analysis and model diagnostics  
   - Implemented nonparametric machine learning models  

4. Model Development  
   - Built and evaluated Gradient Boosting model  
   - Compared performance across different models  

## Main Features  
1. Defect Cause Analysis  
   - Identified key factors influencing scale defects in the rolling process  

2. Model Development  
   - Achieved high predictive performance (Accuracy: 99.5%) using Gradient Boosting  

3. Process Optimization Strategy  
   - Proposed AI-based temperature control and process optimization strategies  

## Key Insights  
- High rolling temperatures (above 1000°C) significantly increase defect rates  
- Cooling and descaling conditions are critical factors affecting defect occurrence  
- Gradient Boosting showed the best predictive performance  

## Expected Impact  
- Enables data-driven quality control in steel manufacturing  
- Supports real-time process optimization  
- Reduces defect rates and improves production efficiency
